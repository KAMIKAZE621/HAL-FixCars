HAL-FixCars は、FiveM（QBCore）向けの
「メカニック不在時のみ使用可能な簡易修理台スクリプト」 です。

メカニックジョブがオンラインにいない場合に限り、
プレイヤーが修理台を使って車両を修理できます。

📦 必須リソース（Dependencies）

以下のリソースが 必須 です。

QBCore

ox_lib

ox_target
